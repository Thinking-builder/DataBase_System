计算机与人工智能领域科研助手 · 使用说明

📌 模块介绍

减论Bot 是为计算机与人工智能领域科研工作者设计的辅助助手，聚焦以下两大核心功能：

  • 论文减论Bot：提供高质量的论文推荐、分析与解读。
  • 资讯减论Bot：推送前沿AI科技资讯、GitHub热榜、全球新闻动态。
🚦 使用逻辑与自动跳转

系统会根据您的提问内容，自动切换到最合适的模块：

跳转至资讯减论Bot的触发条件：

  • 需要 科研资讯 / 技术热点 / 新闻
  • 想了解 GitHub Trending 项目
  • 主动提问：“今天有什么新资讯？”、“AI最近有什么突破？”

跳转至论文减论Bot的触发条件：

  • 需要 计算机 / 人工智能 相关论文推荐
  • 上传了图片、论文截图或完整PDF文档
  • 使用“详细分析”、“逐段阅读”等需求关键词
📚 模块功能一览

论文减论Bot：

  • 推荐高引用、高价值论文
  • 按领域（如深度学习、自然语言处理、强化学习等）精准匹配
  • 逐段分析论文内容，帮助理解复杂方法

资讯减论Bot：

  • 实时更新AI前沿进展与技术应用
  • 聚合GitHub热门开源项目
  • 全球重要科技新闻同步推送
💡 使用小贴士
  • 多领域需求建议分开表达，如“请推荐AI领域+计算机视觉论文”。
  • 默认返回 5条 资讯 / 论文结果，如需更多，请说明“给我10条”等。
  • 遇到“agent调用出错”等提示时，刷新或重新请求即可。
